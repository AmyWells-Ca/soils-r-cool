# soils-r-cool

## About
This repository contains a variety of code that was developed to make it easier to analyze data within the Soils Subcluster as part of the Kwiakah First Nation's research efforts.

By default, xlsx files, csv files, and .png files are not saved in the repository in order to protect data privacy.

## How to Use:
1. Download R (https://www.r-project.org/) and R-Studio (https://posit.co/download/rstudio-desktop/)
2. Open the "soils-r-cool.Rproj" file in the root directory with R Studio
3. Within R-studio, open the r files (located in r/) to edit or run code!
4. Use your favourite git manager (Amy Wells is partial to git-fork; https://git-fork.com/) to clone the repository
5. If you make additions to the code, submit a pull request with your changes
6. Amy will review, and integrate them into the code!
